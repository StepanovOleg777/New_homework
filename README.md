# Описание

Простая и эффективная система управления продуктами и 
категориями на Python с тестированием.

## 🚀 Возможности

- **Управление продуктами**: создание, хранение и управление товарами
- **Управление категориями**: организация продуктов по категориям  
- **Автоматический подсчет**: автоматический учет количества продуктов и категорий
- **Валидация данных**: проверка корректности цен и количеств
- **Наследование классов**: Smartphone и LawnGrass как наследники Product
- **Ограничения сложения**: можно складывать только продукты одного типа
- **Валидация типов**: проверка при добавлении продуктов в категории
- **Расширенные атрибуты**: дополнительные поля для специфичных товаров
- **Тестирование**: покрытие кода тестами
- **Абстрактные классы**: BaseProduct как абстрактный родительский класс
- **Миксины**: LoggingMixin для логирования создания объектов
- **Множественное наследование**: Product наследует от миксина и абстрактного класса
- **Расширенная архитектура**: BaseEntity для общих функциональностей Category и Order
- **Обработка исключений**: ZeroQuantityError для товаров с нулевым количеством
- **Средний ценник**: Метод middle_price() для расчета средней цены в категории
- **Безопасное добавление**: Методы с проверкой и обработкой исключений
- **Try/Except/Finally**: Полноценная обработка ошибок с завершающими действиями
